# AETHER — Sequential Watercolor Generator

Интерактивный генератор акварельных композиций по введённому тексту. Проект рендерит последовательные траектории букв на сетке и рисует их в разных стилях «движков».

## Возможности
- Ввод текста и разбиение по словам.
- Переключение стилей рендера (несколько движков).
- Палитры с категорией и быстрым выбором.
- Экспорт PNG.
- Переключатели: Grid overlay, A‑Z (случайная раскладка), Dark Mode.

## Безопасность (кратко)
- CSP с строгими источниками и SRI для внешних библиотек.
- Нет сетевых запросов, отправки данных или хранения пользовательского ввода.
- Динамическая загрузка движков ограничена белым списком.

## Запуск локально
Простой статический сайт. Достаточно любого локального сервера:
- VS Code: Live Server
- Python: `python3 -m http.server`

Откройте `http://localhost:8000` и убедитесь, что всё работает.

## GitHub Pages
Проект совместим с GitHub Pages, так как использует относительные пути и статические файлы.

1. Включите Pages для ветки `main`.
2. Дождитесь публикации.
3. Откройте выданный URL.

## Структура
- index.html — UI, базовая инициализация и переключатели.
- sketch.js — основной контроллер, взаимодействие с движками.
- diffusion.js — логика «чернил» и выбор палитр.
- engines/ — изолированные движки рендера.
- palettes/ — список палитр.
